-------------------------------
My Rocket Game
-------------------------------

Description:
------------

Its a duel battle between 2 space ship, you vs computer...

How to play:
------------

Open "index.html" in a Web Broswer.

Support:
--------

Runs on any Browser ... recommended: 
=> Chrome 
=> Firefox

Lastest stable as of 2014 or later.


Controls:
---------

Arrow keys
UP   - Thrust
DOWN - Shoot
LEFT - Rotate left
RIGHT- Rotate Right
